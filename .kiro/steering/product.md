# Product Overview

Page Health Analyzer is a lightweight browser-based tool for analyzing webpage performance health against configurable Non-Functional Requirements (NFRs).

## Core Purpose
- Collect comprehensive performance metrics using Web Performance APIs
- Evaluate page health against configurable thresholds  
- Present results in user-friendly format
- Deploy as browser extension or injectable script
- Safe operation on any website using read-only APIs

## Key Features
- **Performance Monitoring**: Navigation timing, resource metrics, rendering performance, network activity
- **Health Assessment**: Configurable thresholds with PASS/WARN/FAIL status
- **Dual Deployment**: Browser extension and injectable script modes
- **Safe Operation**: Read-only APIs, no website modification

## Target Users
Developers and performance engineers who need quick, configurable webpage performance analysis without complex tooling setup.
